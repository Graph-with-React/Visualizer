This tool will be helpful to visualize the Prims MST Algorithm. While a graph has been provided you can generate a new graph with the number of nodes desired, by writting an input and clicking generate graph.

Instructions

Nodes will be added if you click "Add new node". You can add and update edges by selecting a valid starting and ending node as well as a weight from 1-20 inclusive. Currently there is not way tp delete edges, if you prefer to create your own graph create a graph with 1 node and add from there. Currently the generate graph will generate up to 15 nodes.
